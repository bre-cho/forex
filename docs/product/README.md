# Product Documentation

## Features

### Trading Bots
- Create bots per workspace
- Configure symbol, timeframe, risk parameters
- Start/stop/pause/resume in real-time
- Paper trading and live cTrader execution

### Elliott Wave Engine
- Automatic wave detection (Elliott 1-5, A-B-C)
- Wave confidence scoring
- Retracement level analysis

### AI Features (Pro+)
- LLM signal confirmation (OpenAI / Anthropic)
- Multi-agent decision engine
- Adaptive risk management

### Analytics
- Equity curve
- Win rate, profit factor, Sharpe ratio
- Drawdown analysis
- Trade history

### Notifications
- Email, Telegram, Discord, custom webhook
- Bot status changes
- Trade execution alerts
- Drawdown warnings

## User Guide

1. Register and create a workspace
2. Add a broker connection (paper or live cTrader)
3. Create a strategy
4. Create a bot, configure risk settings
5. Start the bot and monitor via dashboard
